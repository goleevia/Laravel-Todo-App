# Laravel Todo App (Laravel 12 + Vue 3 + InertiaJS)

## Features
- Create Todo
- Read Todos
- Update Todo
- Delete Todo
- Validation
- PHPUnit tests

## Tech Stack
- Laravel 12
- Vue 3
- InertiaJS
- MySQL

## Setup

composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
npm run dev
php artisan serve

## Run Tests

php artisan test
